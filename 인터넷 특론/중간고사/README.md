## 1. 이동통신에서 다중액세스(MA: Multiple Access) 기술의 목적 및 원리를 설명하고, 1세대에서 3세대까지 세대별로 대표적인 MA 기술을 설명하시오. 

목적 : 여러 사용자가 하나의 주파수 자원을 효율적으로 공유하기 위함
<br>
원리 : 자원을 나눠서 사용함
- 시간 / 주파수 / 코드 등으로 분할

<br>

- 1세대 : FDMA 주파수를 분할하여 사용
- 2세대 : TDMA 시간을 분할하여 사용
- 3세대 : CDMA 각 사용자에게 코드를 부여

## 2. 이동통신은 셀룰러 기반으로 전체 담당 지역에 서비스를 제공하는데, 셀룰러 방식의 개념을 설명하시오. 

개념
- 전체 서비스 지역을 여러 개의 셀(Cell)로 분할
- 각 셀마다 기지국(Base Station) 배치

<br>

핵심 특징
- 주파수 재사용(Frequency Reuse) 가능
- 이동 중에도 핸드오프(Handoff)로 서비스 유지
- 시스템 용량 증가

<br>

요약 : 서비스 지역을 여러 셀로 분할하여 셀마다 기지국을 배치, 서로 간섭이 일어나지 않도록 하는 방식

## 3. 무선 LAN에서 여러 가지 전송방식 중, 적외선(IR) 통신 방식의 장단점을 일반적인 RF 전파 방식과 비교하여 열거하시오. 

### IR 방식

장점
- 전파 간섭 없음
- 설계가 쉬움
- 제조 단가 저렴
- 전력 소모가 적음

<br>

단점
- 장애물 통과 불가
- 일직선으로 통과

<br>

### RF 방식

장점
- 장애물 통과 가능
- 장거리 통신 가능
- 이동성 좋음

<br>

단점
- 전력 소모가 많음
- 설계가 어려움
- 간섭 발생 가능 

## 4. IEEE LLC(Logical Link Control) 링크 서비스는 현재 3가지 종류로 구분할 수 있는데, 특히 무선 LAN에서 어떠한 종류의 서비스가 필요한지 유선 LAN과 비교하여 설명하시오. 

3가지 서비스
- 연결형 서비스
- 비연결형 서비스
- 비연결 확인형 서비스

<br>

유선 LAN
- 오류 발생 적음 (안정적)
- 비연결형 서비스 사용
- 확인 절차 없이 빠르게 전송

<br>

- 무선 LAN
- 간섭, fading 등으로 오류 많음
- 비연결 확인형 서비스 필요 (ACK 기반)

<br>

요약 : 무선 LAN은 오류가 많아 확인 기능이 있는 서비스가 필요하고, 유선 LAN은 안정적이므로 비연결형 서비스 사용

## 5. 무선 LAN의 DSSS(Direct-Sequence Spread Spectrum) 방식과 이동통신의 CDMA(Code Division Multiple Access) 방식과의 차이점을 설명하고, 또한 DSSS 전송 방식인 IEEE 802.11b의 경우 어떻게 데이터 전송률을 1Mbps로부터 11Mbps까지 개선하였는지 원리를 설명하시오. 

DSSS vs CDMA 차이점
<br>

#### DSSS

- 데이터에 확산 코드(Barker sequence 등)를 곱해 신호를 넓은 대역으로 확산
- 목적 : 잡음 및 간섭에 강하게
- 전송 방식임

<br>

특징
- 한 사용자의 신호를 확산하는 기술
- 다중접속 자체가 목적은 아님

#### CDMA

- 각 사용자에게 고유 코드(직교 코드)를 부여
- 같은 주파수/시간에서 동시에 전송 가능
- 다중접속 방식

<br>

특징
- 여러 사용자가 동시에 통신 가능
- 코드로 사용자 구분

<br>

#### 원리

CCK를 이용하여 한 심볼당 비트 수를 증가시켜 데이터 전송률 향상

## 6. OFDM(Orthogonal Frequency Division Multiplexing) 기반의 IEEE 802.11a 무선 LAN 방식에서 최소 데이터 전송률을 계산하시오. ( 문제 비교 참고#6 ) (단, Baud rate = 250 K, # of Sub-carrieres = 48, Modulation = BPSK, Convolutional coding rate = 1/2) 

조건
- Baud rate = 250 K
- 서브캐리어 수 = 48
- 변조 방식 = BPSK
- 부호화율 = 1/2

<br>

결과
- 서브캐리어 48개 × BPSK(심볼당 1비트) × 부호화율 1/2 × 심볼율 250K
